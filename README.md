# Budget Calculator

This repo can be published as a static GitHub Pages site.

## Publish On GitHub Pages

1. In GitHub, open `Settings` -> `Pages`.
2. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
3. Save, then wait for GitHub to publish the site.
4. Share the GitHub Pages URL with friends.

## Notes

- The published site uses browser `localStorage`, so each person keeps their own saved entries in their own browser.
- CSV import/export works on GitHub Pages because it is fully client-side.
- The local Node server is only needed for local development.
